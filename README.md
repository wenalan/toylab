使用方法：
点击地图添加多个地点
之后会自动根据公交线路安排一个合理路线，一次访问所有地点

已知问题
1. 虽然会用每个地点作为起点进行尝试，最后选定一个距离最短的方案进行展示。但是实际上，展示的路径是公交路径，但是比较时的路径是直线距离。这样比较速度非常快，但是结果并不准确
2. 点击分析时，会逐个计算每个地点开始的公交路线，所以速度比较慢，但是路径更准确一些。这里选出的路径和上面的结果可能不一样
3. 选取下一个点的时候，是通过直线距离最短的方式，用贪心法挑选的，所以全局结果不一定是最优的。特别是两个点靠的很近，但是实际公交路线，因为河流，铁路，需要绕很远的情况。
